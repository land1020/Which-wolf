# 人狼ドッチ ゲームリファレンス

『人狼ドッチ』は、配られた2枚のカードから自分の役職を「ドッチ（どっち）」にするか選ぶ、スピーディな正体隠匿ゲームです。

---

## 1. 概要とコンセプト

| 項目 | 内容 |
|------|------|
| ジャンル | 人狼 |
| プラットフォーム | ブラウザ（HTML/JS） |
| フレームワーク | antigravity |
| 通信方式 | ローカルテスト → オンライン（最終目標） |
| プレイ人数 | 最大8人（ロビー設定で変更可） |

### ゲームの基本フロー

```
[全員がロビーに集合]
  ↓ ホストが役職の組み合わせ・議論時間を設定
  ↓ 各プレイヤーがプレイヤーカラーを選択

[ゲーム開始]
  ↓ 配布と選択（ドッチにする？）
      各プレイヤーに2枚ずつカードを配布します（総カード数 = プレイ人数 × 2）。
      中身を確認し、どちらか1枚を「役職カードエリア」に、もう1枚を「残りカードエリア」に伏せて置きます。

[準備フェーズ]
  ↓ 能力を持つ各役職が並列で能力を使用します（夜能力：人狼・大狼・少年・占い師）。
    ・能力持ちプレイヤーにのみ操作UIを表示、他プレイヤーは待機表示。
    ・能力を使用せず「完了」ボタンを押すことも可能。
    ・全員が「完了」を押したら次のフェーズへ。

[朝フェーズ]
  ↓ 全員で議論を行います。（デフォルト3分間タイマー）
    ・時間経過後、自動で昼フェーズへ移行。

[昼フェーズ]
  ↓ 能力を持つ各役職が順番に能力を使用します（昼能力：情報屋 → 警察 → 怪盗 → DJ）。
    ・能力持ちプレイヤーにのみ操作UIを表示、他プレイヤーは待機表示。
    ・能力を使用せず「完了」ボタンを押すことも可能。
    ・対象プレイヤーが「完了」を押したら次の役職の処理へ。

[夜フェーズ]
  ↓ 全員で再度議論を行います。（デフォルト3分間タイマー）
    ・時間経過後、自動で投票フェーズへ移行。

[投票フェーズ]
  ↓ 全員が怪しいプレイヤーへ一斉投票します。
    ・自分への投票は不可。
    ・「平和村へ投票」の選択肢あり（ノーリンチ選択）。
    ・全員が投票先を決めたら次の処理へ進む。

[処刑判定]
  ↓ 最も票を集めたプレイヤーが追放されます。
    ・同数票の場合は、該当する全プレイヤーが追放されます。
    ・全員が「平和村へ投票」を選択し、かつ全員が村人陣営の場合は全員勝利。

[勝敗判定]
  ↓ 勝利条件の判定を行います。

[リザルト画面]
  ↓ 各プレイヤーの役職・結果を表示。罠師の道連れ結果もここで表示。
```

### コンセプトの要点

- 人狼ドッチという既存ゲームをWEBで友人と遊べるようにゲーム開発を行う

---

## 2. 勝敗条件

- **人間チーム**: 追放会議で「人狼」を1人でも追放できれば勝利。
- **人狼チーム**: 誰一人として人狼が追放されなければ勝利。
- **第3陣営**: 各役職固有の勝利条件を達成すれば単独勝利。

---

## 3. 役職システム

パッケージごとの主な収録役職です。

### ■ 基本セット（通常版/ホワイト・ブルー）

| 役職名 | 陣営 | 能力使用フェーズ | 能力・特徴 |
| :--- | :--- | :--- | :--- |
| **市民** | 人間 | なし | 特殊能力なし。 |
| **占い師** | 人間 | 準備フェーズ | 「誰かの役職カード」または「誰かの残りカードエリアから2枚」を確認する。 |
| **警察** | 人間 | 昼フェーズ | 「誰かの残りカード」を1枚確認する。 |
| **DJ** | 人間 | 昼フェーズ | 「誰かの残りカード」と「別の誰かの残りカード」を入れ替える。自分は対象に選べない。 |
| **人狼** | 人狼 | 準備フェーズ | 仲間の人狼（人狼・大狼）が誰かを確認する。 |
| **裏切り者** | 人狼 | なし | 能力なし。人狼チームが勝利すれば勝利。 |
| **おばけ** | 第3陣営 | なし | 自分が追放されたら単独勝利。 |

### ■ 拡張セット（レッド）

| 役職名 | 陣営 | 能力使用フェーズ | 能力・特徴 |
| :--- | :--- | :--- | :--- |
| **大狼** | 人狼 | 準備フェーズ | 仲間の人狼（人狼・大狼）が誰かを確認する。占い師に占われても「市民」と判定される。 |
| **少年** | 人間 | 準備フェーズ | 仲間の「少年」が誰かを確認できる。 |
| **怪盗** | 人間 | 昼フェーズ | 「誰かの役職カード」と「自分の役職カード」を入れ替える。入れ替え後、相手の（元の）役職を確認できる。 |
| **情報屋** | 人間 | 昼フェーズ | 「誰か2人の残りカード」をそれぞれ1枚ずつ確認する（警察の上位互換）。 |

| **逃亡者** | 第3陣営 | なし | 自分が追放されなければ、人間・人狼どちらが勝っても勝利。 |
| **罠師** | 人間 | なし | 自分が追放されると、自分に投票したプレイヤーを道連れにする。道連れにされたプレイヤーはリザルト画面で「道連れ」と表示され、敗北扱いとなる。 |

---

## 4. 能力処理の詳細

### 準備フェーズ（並列処理）

能力持ちの全役職が同時に能力を使用します。カードの移動は発生しないため、順番は不要です。

| 役職 | 処理内容 |
| :--- | :--- |
| 人狼・大狼 | 人狼陣営（人狼・大狼）のプレイヤー名を全員表示する |
| 少年 | 少年役職を持つプレイヤー名を全員表示する |
| 占い師 | 対象プレイヤーの役職カード、または指定した残りカード2枚を表示する |

### 昼フェーズ（順番処理）

カードの移動が発生するため、以下の順番で処理します。

| 順番 | 役職 | 処理内容 |
| :--- | :--- | :--- |
| 1 | 情報屋 | 対象プレイヤーの残りカードエリアのカードをすべて表示する |
| 2 | 警察 | 対象プレイヤーの残りカードを1枚表示する |
| 3 | 怪盗 | 対象プレイヤーの役職カードと自分の役職カードを入れ替える。入れ替え後、相手の元の役職を確認する |
| 4 | DJ | 対象プレイヤーAの残りカードと対象プレイヤーBの残りカードを入れ替える |

---

## 5. ロビー設定

### ホストが設定できる項目

| 設定項目 | 内容 |
| :--- | :--- |
| 役職の組み合わせ | 使用する役職を自由に選択。人狼が含まれていない場合はエラーを表示する |
| 議論時間 | 朝フェーズ・夜フェーズの議論時間（デフォルト：各3分） |

### プレイヤーが設定できる項目

| 設定項目 | 内容 |
| :--- | :--- |
| プレイヤーカラー | 自分の名前背景（プレイヤーマット）の色を選択する |

---

## 6. 特殊なケース

- **平和村**: 全員が「平和村へ投票」を選択し、かつ参加プレイヤー全員が村人陣営（人間チーム）の場合、全員勝利となります。
- **同数票**: 投票で最多票が同数だった場合、該当する全プレイヤーが追放されます。
- **罠師の道連れ**: 罠師が追放された場合、罠師に投票した全プレイヤーが道連れになります。リザルト画面に表示され、敗北扱いとなります。
- **2人プレイ**: 特殊ルールとして「場」のカードを推測する独自の遊び方になります。（詳細は別途定義）
